Will add a shaped crafting recipie to the game

Parameters
recipieThe desired recipie. Layout is {"XXX", "XXX", "XXX", "X", ItemID} where each X is a slot in the crafting grid, Each group of 3 is a row, and a "X", ItemID is the Item ID X represents (for each new item a new symbol is required), a Sapce is no item required in that slot
resultThe Item that the recipie will produce

This example shows how to call AddShapedRecipie(object[], Item)

void Main()
{
CraftingRecipies.AddShapedRecipie(new object[] { " X ", "X@X", " X ", "X", Wood.GetItemID(), "@", Stone.GetItemID() }, new Chest());
}
